Lead Magnet Builder - Ressources de capture

Quand utiliser
- Creer un guide PDF, checklist ou mini-outil pour capturer des emails
- Produire une ressource gratuite qui demontre l'expertise EasyClaw
- Construire la base email pour le nurturing Brevo

Prompt systeme

Tu crees des lead magnets pour PME francaises. Chaque ressource doit apporter une valeur immediate et concrete - le prospect doit pouvoir l'utiliser le jour meme. Le lead magnet est une preuve de competence, pas un teaser.

Regles metier VibeCraft

Types de lead magnets
| Type | Format | Effort | Conversion |
|---|---|---|---|
| Checklist | PDF 1-2 pages | Faible | Haute |
| Guide pratique | PDF 5-10 pages | Moyen | Haute |
| Template/Modele | Doc/Sheet editable | Moyen | Tres haute |
| Mini-audit | Formulaire + resultat | Eleve | Tres haute |
| Demo live | Scraping/scoring en direct | Eleve | Maximum |

Structure guide PDF
1. COUVERTURE - Titre benefice, sous-titre cible, logo EasyClaw
2. INTRO (1 page) - Le probleme, pour qui, ce qu'ils vont apprendre
3. CONTENU (3-7 pages) - Etapes numerotees, exemples concrets FR
4. CHECKLIST RECAP (1 page) - Actions a cocher
5. A PROPOS + CTA (1 page) - EasyClaw en 3 lignes + lien


Regles absolues
1. Valeur immediate - utilisable le jour meme, pas "contactez-nous pour la suite"
2. Specifique metier - un lead magnet par verticale, pas un guide generique
3. Zero chiffre invente - jamais "gagnez 10h/semaine". Dire "reduisez le temps passe sur X"
4. RGPD - mention legale obligatoire sur le formulaire de capture
5. Branding leger - EasyClaw visible mais pas omnipresent

Lead magnets prioritaires par verticale

Comptables :
- "Checklist : les 7 taches de cabinet a automatiser en premier"
- "Guide : mettre en place un agent de relances impayees en 1 journee"

Avocats :
- "Template : grille de tri automatique des demandes entrantes"
- "Guide : veille juridique automatisee sans y passer 2h/jour"

Gestion locative :
- "Checklist : automatiser la gestion locative etape par etape"
- "Template : modele de relance loyer impaye personnalisable par agent IA"

RH / Recrutement :
- "Guide : pre-qualifier les candidatures avec un agent IA"
- "Template : scoring candidat automatise - criteres et ponderation"

Exemple concret

Cible : Comptables independants / petits cabinets
Lead magnet : "Checklist : les 7 taches de cabinet a automatiser en premier"
Contenu :
1. Relances clients impayees - agent envoie email J+7, J+15, J+30
2. Tri des pieces comptables entrantes - classification auto par type
3. Veille reglementaire - resume quotidien des changements
4. Reponses FAQ clients - agent repond aux questions recurrentes
5. Suivi des echeances fiscales - alertes automatiques
6. Pre-saisie comptable - extraction donnees factures
7. Reporting client mensuel - generation automatique

Pour chaque tache : ce que l'agent fait, ce qui reste humain, temps estime recupere (fourchette qualitative, pas de chiffre precis invente).

Distribution : Landing page geteasyclaw.app/guide-comptable - Brevo formulaire - envoi auto PDF

Process
1. Identifier verticale + douleur principale
2. Choisir format (checklist souvent le meilleur ratio effort/conversion)
3. Rediger contenu actionnable
4. Verifier : valeur immediate, zero invention, RGPD
5. Sauver dans ~/workspace/content/lead-magnets/
6. Validation Mehdi avant mise en ligne

Pitfalls
- Un lead magnet trop long ne sera pas lu. 2-5 pages max pour une checklist
- Ne pas promettre de resultats chiffres qu'on ne peut pas garantir
- Le formulaire Brevo doit avoir : consentement explicite, lien desinscription, mention CNIL
